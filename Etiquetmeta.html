TEMA 1

Etiquetas de metadatos

  • <head> : Representa una colección de metadatos (datos sobre datos) para el documento.
  • <title> : Define el título de la página.
  • <base> : Indica la dirección de inicio de la web, con el uso del atributo href, para el uso de direcciones relativas.
  • <link rel> : Se utiliza para adjuntar documentos externos como el de CSS o JavaScript.
  • <style> : Etiqueta para introducir CSS en el mismo documento, aunque es preferente utilizar archivos adjuntos.
  • <script> : Etiqueta para introducir JavaScript en el mismo documento, aunque es preferente utilizar archivos adjuntos.
  • <meta> : Se utiliza para añadir Información sobre la web, utilizado por los buscadores para el posicionamiento, como:
    • <meta charset=”UTF8”> : Conjunto de caracteres utilizado en la web.
    • <meta name=”description” content=”” > : Descripción breve del la web.
    • <meta name=”keywords” content=”” > : Palabras clave para la busqueda.
    • <meta name=”author” content=””> : Autor.
    • <meta http-equiv=”refresh” content=”"> : Tiempo de refresco en segundos.
    • <meta name=”viewport” content=”width-device-width, initial-scale=1.0> : Da las instrucciones al navegador para controlar las dimensiones y la escala de la página.
