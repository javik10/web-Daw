TEMA 1

Elemento de línea

  • <a> : Para añadir enlaces.
  • <small> : Hace el texto mas pequeño.
  • <cite> : Marca una referencia a una fuente.
  • <q> : Indica que el texto adjunto es una cita corta en línea.
  • <code> : Para marcar el código de un programa.
  • <sub> : Se utiliza para subíndices.
  • <sup> : Se utiliza para superíndices.
  • <span> : Utilizado para agrupar en línea y para poder dar formato desde CSS.
  • <br> : Para realizar un salto de línea.
  • <wbr> : Una posición dentro del texto donde el explorador puede opcionalmente saltar una línea.
  • <em> : Para marcar con enfasis partes del texto.
  • <strong> : Marcar las partes mas importantes del texto.
