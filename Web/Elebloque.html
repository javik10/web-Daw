TEMA 1

Elemento de bloque

  • <p> : Etiqueta para escribir texto en parrafos. El formato se da desde archivo CSS actualmente, por lo que los atributos para dar formato estan desfasados.
    • <p> parrafo de ejemplo </p>
  • <address> : Para la informacion de contacto para el article asociado. No esta permitido que el contenido sea un address anidado, contenido de cabecera, de sección ni elmento header o footer.
    • <address> You can contact author at <a href="http://www.ejemplodecontacto.com">www.contacto.com</a> </address>
  • <hr/> : Se utiliza para separar parrafos. Es recomendoble nop utlicizar las etiquetas asociadas para darle formato y en su lugar hacerlo con CSS.
    • <p> Parrafo visual para ser separado </p> <hr/> <p> Parrafo visual para ser separado </p>;
  • <h1>-<h1> : Uso para titulos, letras en negrita y tamaño dependiendo del valor. El formato se da desde archivo CSS actualmente, por lo que los atributos para dar formato estan desfasados.

    • Ejemplo h3

    Ejemplo h4

    Ejemplo h5
  • <pre> : Texto con formato preformado, se muestra tal y como esta escrito en el archivo html. Tan solo tiene atributos globales.
    • 											ejemplo de 
											texto preformateado
  • <blockquote> : Se utiliza para añadir citas a otros autores o documentos. Como atributo especifico tiene "cite", el cual proporciona enlace a la fuente original.
    • <blockquiote cite="http://www.ejemplodecita.com"> Contenido de la cita. </blockquote>
  • <ol> : Para hacer listas ordenadas, la lista puede estar ordenada de forma inversa si se utiliza reversed. Actualmente el formato se le da con el uso de CSS.
    • <ol>
     <li>primer objeto</li>
     <li>segundo</li>
     </ol>
  • <ul> : Para hacer listas no ordenadas. Actualmente el formato se le da con el uso de CSS.
    • <ul>
     <li>primer objeto</li>
     <li>segundo</li>
     </ul>
  • <dl> : Para listas descriptivas. Actualmente el formato se le da con el uso de CSS.
    • <dl>
     <dt>palabra a definir</dt>
     <dd>definicion</dd>
     </dl>
  • <div> : Utilizado para agrupar en bloques y para poder dar formato desde CSS.
    • <div> Contenido del bloque </div>
  • <img> : Para introducir imagenes, introduciendo enlaces con src. Consta del atributo "alt" el cual muestra un texto alternativo en caso de que no se cargue la imagen, esta informacion puede ser complementada con "longdesc". Otro atributo es "crossorigin" El cual indica si la buesqueda de la imagen debe de ser CORS, y los valores permitidos son "anonymous" en caso de no enviar credenciales o "use-credentials" en caso de enviarlas.
    • <img src="head.PNG" id="cabecera"></img>
  • <iframe> : Permite incrustar otro documento html con el atributo src. No se utiliza mucho hoy en dia.
    • <iframe src="footer.html"></iframe>
