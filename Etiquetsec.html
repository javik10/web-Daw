TEMA 1

Etiquetas de sección

  • <body> : Es el cuerpo de la web, donde se va a encontrar todo el contenido de esta misma que se mostrara al usuario.
  • <header> : Se encarga de de indicar queel contenido se trata de una cabecera .
  • <article> : Representa un contenido completo y diferenciado. Se recomienda que dentro exista un encabezado.
  • <h1>-<h6> : Se utiliza para títulos, conforme mayor es el número menor la importancia. Un elemento de encabezado describe brevemente el tema de la sección que presenta.
  • <section> : Divide el contenido, esta pensado para separar contenido conectado pero diferenciado.
  • <aside> : Representa un contenido relacionado con el contenido principal. Usado para menús laterales.
  • <footer> : Pie de página, utilizado para informacion del creador, contacto,localizacion o enlaces relacionados.
  • <nav> : Indica una sección de navegación como un menu.
  • <main> : Representa el contenido principal de la web o aplicación. Debe de ser unico al documento y excluir contenido que se repita en otros documentos como pie de página o barras de navegación
