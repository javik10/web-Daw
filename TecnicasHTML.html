Tecnicas HTML

H49 Uso de marcado semántico para marcar texto especial o enfatizado

El objetivo de esta técnica es demostrar cómo se puede utilizar el marcado semántico para marcar texto especial o enfatizado de modo que se pueda determinar mediante programación. El uso de marcado semántico para marcar texto enfatizado o especial también proporciona estructura al documento.

La mayoría de los agentes de usuario distinguen visualmente el texto que se ha identificado mediante el marcado semántico.

H51 Usar marcado de tabla para presentar información tabular

El objetivo de esta técnica es presentar la información tabular de una manera que preserve las relaciones dentro de la información incluso cuando los usuarios no pueden ver la tabla o se cambia el formato de presentación. Es decir no centrarse únicamente en como se muestra visualmente si no en la relación que hay entre los elementos. de esta forma, aunque visualmente utilizando la etiqueta pre, podemos mostrar una tabla sin bordes, se debe de utilizar las etiquetas de table para dar valor semántico y ayudar aquellos que no pueden ver a recibir la información de forma correcta.

cabecera
celda 1 celda 2 
                          cabecera
                            celda 1   celda 2

H53 Usando el cuerpo del elemento object

El objetivo de esta técnica es proporcionar una al texto para el contenido renderizado utilizando el elemento de object. El cuerpo del elemento object se puede utilizar para proporcionar una alternativa de texto completo para el objeto, o contener contenido adicional que no sea texto como alternativas.

H54 Usar el elemento dfn para identificar la instancia definitoria de una palabra

El objetivo de esta técnica es utilizar el dfn para marcar el uso de una palabra o frase donde se defina. El elemento dfn se utiliza para indicar la instancia definitoria del término adjunto. Tenga en cuenta que incluye el término, no la definición.

Ejemplo: El muro de Berlin fue parte de la frontera interalemana desde el 13 de agosto de 1961 hasta el 9 de noviembre de 1989

El objetivo es identificar los cambios de dirección del texto, como pueden ocurrir en caso de utilizar citas o referencias en otros idiomas los cuales se escriben siguiendo otra orientación.

Ejemplo: cambiamos la dirección texto con la dirección cambiada.

H57 Usando el atributo de idioma en el elemento HTML

El objetivo de esta técnica es identificar el idioma predeterminado de un documento proporcionando el atributo lang en el elemento html.

H58 Usar atributos del lenguaje para identificar cambios en el lenguaje humano

El objetivo de esta técnica es identificar claramente cualquier cambio en el idioma de una página mediante el uso del atributo lang. El uso del código principal es importante para esta técnica. El uso de subcódigos es opcional, pero puede resultar útil en determinadas circunstancias.

H59 Usar el elemento de enlace y las herramientas de navegación

El objetivo de esta técnica es describir cómo el elemento link puede proporcionar metadatos sobre la posición de una página HTML dentro de un conjunto de páginas web o puede ayudar a localizar contenido con un conjunto de páginas web. El valor de los rel atributos indica qué tipo de relación se está describiendo y el href atributo proporciona un enlace al documento que tiene esa relación. Múltiples elementos links pueden proporcionar múltiples relaciones. Varios valores de rel son útiles:

  • Inicio: se refiere al primer documento de una colección de documentos.
  • Siguiente: se refiere al siguiente documento en una secuencia lineal de documentos.
  • Anterior: se refiere al documento anterior en una serie ordenada de documentos.
  • Contenido: se refiere a un documento que sirve como tabla de contenido.
  • Índice: se refiere a un documento que proporciona un índice para el documento actual.

H60 Usar el elemento de Link para enlazar a un glosario

El objetivo de esta técnica es proporcionar un mecanismo para localizar un glosario. Cuando los términos del contenido se definen en una página de glosario separada, se hace referencia al glosario utilizando un elemento link en el head. El atributo rel del elemento link se establece en "glosario" y el atributo href contiene el URI de la página del glosario.

H65 Usar el atributo title para identificar controles de formulario cuando no se puede usar el elemento de etiqueta

El objetivo de esta técnica es utilizar el atributo title para proporcionar un nombre accesible para los controles de formulario cuando el diseño visual no incluye texto en la pantalla que pueda asociarse con el control como una etiqueta. La tecnología de asistencia, pueden nombrar el atributo title.
