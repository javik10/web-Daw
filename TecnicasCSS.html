Tecnicas CSS

C13 Usar tamaños de fuente con nombre

El objetivo de esta técnica es especificar un tamaño de fuente con nombre que exprese el tamaño de fuente relativo deseado. Estos valores proporcionan sugerencias para que pueda elegir un tamaño de fuente.

C14 Uso de unidades em para tamaños de fuente

El objetivo de esta técnica es especificar el tamaño de fuente del texto en unidades em para que los agentes de usuario puedan escalar el contenido de manera efectiva. Dado que el em es una propiedad de la fuente, se escala a medida que la fuente cambia de tamaño.

C15 Usar CSS para cambiar la presentación de un componente de la interfaz de usuario cuando recibe el foco

El objetivo de esta técnica es demostrar cómo se puede mejorar la apariencia visual a través de hojas de estilo para proporcionar retroalimentación visual cuando un elemento interactivo tiene el foco o cuando un usuario se desplaza sobre él usando un dispositivo señalador. Resaltar el elemento que tiene el foco o sobre el que se desplaza el cursor puede proporcionar información como el hecho de que el elemento es interactivo o el alcance del elemento interactivo.

C17 Escalar elementos de formulario que contienen texto

El objetivo de esta técnica es garantizar que los controles de formulario basados ​​en texto cambien de tamaño cuando se cambie el tamaño del texto. Esto permitirá a los usuarios introducir texto y leer lo que introdujeron en los cuadros de entrada porque el texto se muestra en el tamaño requerido por el usuario.

C18 Uso de reglas de margen y relleno de CSS en lugar de imágenes espaciadoras para el diseño del diseño

Los diseñadores web a veces usan imágenes con espaciador (generalmente de 1x1 píxeles, GIF transparentes) para un mejor control sobre el diseño. Sin embargo, las hojas de estilo en cascada (CSS) permiten un control suficiente sobre el diseño para reemplazar las imágenes espaciadoras. Las propiedades CSS para márgenes y relleno se pueden utilizar solas o en combinación para controlar el diseño.
