Tecnicas PDF

No actualizar las alternativas de texto cuando se producen cambios en el contenido que no es de texto

Esto provoca que el texto alternativo difiera del contenido al que esta unido

Abrir ventanas que no son solicitadas por el usuario

Las nuevas ventanas desvían el foco de lo que el usuario está leyendo o haciendo. Esto está bien cuando el usuario ha interactuado con una parte de la interfaz de usuario y espera obtener una nueva ventana, como un diálogo de opciones.

Reproducir un sonido de más de 3 segundos donde no hay ningún metodo para detenerlo

Pueden existir sitios web que por ejemplo utilicen musica de fondo para su sitio web, pero siempre se le debe de dar la posibilidad al usuario de detenerlo por si puede generarle algun tipo de problema o molestia, como en caso de necesitar un lector de pantalla.

Especificar colores de primer plano sin especificar colores de fondo o viceversa

Los usuarios con pérdida de visión o problemas cognitivos, de lenguaje y de aprendizaje a menudo prefieren combinaciones específicas de colores de primer plano y de fondo. En algunos casos, a las personas con baja visión les resultará mucho más fácil ver una página web con texto blanco sobre un fondo negro, y es posible que hayan configurado su asestente para que presente este contraste.

El título de una página web que no identifica el contenido

Que el titulo de la página no concuerde correctamente con el contenido puede crear confusioón y molestia en los usuarios.

Usar solo un símbolo gráfico para transmitir información

Utilizar un simbolo puede ser dificil para algunos usuarios para comprender su significado y utilida, ademas los lectores de pantalla pueden tener dificultades a la hora de determinar el significado de este.

Usar caracteres de espacio en blanco para controlar el espaciado dentro de una palabra

El uso de espacios en blanco, como espacio, tabulación, salto de línea o retorno de carro, para formatear visualmente palabras individuales es un fallo ya que cuando se insertan caracteres en blanco para controlar el espacio entre letras dentro de una palabra, pueden cambiar la interpretación de la palabra o hacer que no se reconozca como una sola palabra.
