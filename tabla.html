TEMA 1

Se usa para crear tablas. Dentro se usan colgroup para agrupar columnas y darle estilos, caption para añadir un tilulo en la parte superior de la tabla,thead para contener la fila de la cabezera, tbody para el contenido de la tabla y tfoot para el pie de tabla. Dentro de las tres últmas se introduciría el tr para crear un fila y dentro de tr va td para crear cada celda de esa fila.
